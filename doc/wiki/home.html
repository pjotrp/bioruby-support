bioruby-testing-central is the github test branch of the official Bioruby project. The central ‘master’ tree can be found at: http://github.com/pjotrp/bioruby/tree/master. See the Road map for more information.

The convention is to name your repository as ‘bioruby-testing-yourname’. So my version will be bioruby-testing-pjotr (where I am doing the microarray stuff) and when I have something ready I can push it into ‘master’. If you register yourself with github I can add you as a collaborator.

Note: we are not competing with the main Bioruby tree – this is merely a facility to encourage code submissions and to share early evolutionary development. It is up to the Bioruby maintainers whether any stuff actually gets included in the main tree. This is a bioruby-testing tree.

Clone the central repository with:

git clone git://github.com/pjotrp/bioruby.git

You don’t need to register for that. Patches can be submitted over E-mail. The first thing to do is create your own branch:

git branch bioruby-testing-yourname
git checkout bioruby-testing-yourname

modify, play and commit to your branch. To push it as a separate new branch to github you can:

git push origin bioruby-testing-yourname:refs/heads/bioruby-testing-yourname

So, for example, my tree you can view at http://github.com/pjotrp/bioruby/commits/bioruby-testing-pjotr

For using git see the git tutorial and the Github cheat sheet
